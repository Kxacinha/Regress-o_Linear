# Regressao_Linear


Utilizando regressão linear da biblioteca do ScikitLearn para predizer valores dos imóveis com base no banco de dados adquirido.

Primeiro foi utlizado a biblioteca Pandas para o tratamento dos dados. Depois foi usando a biblioteca do Matplotlib para o data visualization.

Após o tratamento e visualização dos dados obtiddos, foi utilizado o train test split para o teste do modelo e após obter uma acuracidade de 91%, foi usado o LinearRegression para a implemetação do modelo de Regressão Linear.
